Per executar per linia de comandes:

-Depen de la config de l'ordinador on ho anem a fer (de vegades cal), afegir a variables del sistema la ruta del nostre JDK
-En el cmd colocar-nos a la carpeta on estiguin els arxius java amb  comanda "cd RUTA"
-Comanda: "javac *.java" (per compilar, crear l'arxiu .class, de tots els arxius acabats en .java)
-Si la classe que volem executar NO està dins d'un package amb fer: "java NomClasse" seria suficient (NO CAL POSAR .CLASS)
-Si la classe que volem executar SI està dins d'un package cal anar al directori on està el package i fer: "java nomPackage.nomClasseMain"
-Per passar arguments, es passen just despres d'executar el programa
